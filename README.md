PYTHON TO-DO LIST APPLICATION
A simple command-line To-Do List applicationdeveloped in python that helps users organize and manage their daily tasks efficiently. The application provides an interactive menu-based interface where users can add, view, complete, and delete tasks directlt from the terminal.

FEATURES 
-> View all tasks along with their completion status
-> Add new tasks to the to-do list
-> Mark tasks as completed
-> Delete tasks that are no longer needed
-> Simple and user-friendly command-line interface
-> Input validation to handle invalid task selections

HOW IT WORKS
The application stores tasks in a Python list, where each task is represented as a dictionary containing:
-> Task description (text)
-> Completion status (done)
Users interact with the application through a menu-driven system that continuously runs until the user chooses to exit.

TECHNOLOGIES USED 
-> Python 3
-> Lists and dictionaries
->Functions
-> Loops and Conditional Statements
-> User Input Handling

LEARNING OUTCOMES
This project was created to practice fundamental Python Programming  concepts, including:
-> Function creation and modular programming
-> Data structure manipulation
-> User interaction through the command line
-> Input validation and error handling
-> Program flow control using loops and conditionals
